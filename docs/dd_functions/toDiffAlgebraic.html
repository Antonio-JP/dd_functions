ajpastor.dd_functions.toDiffAlgebraic.infinite_derivative(element, times=1, derivation=None)¶

Method that computes the times-th derivative of a differential polynomial.

Given a differential ring \((R,\partial)\), the ring of differential polynomials (denoted by \(R\{y\}\)) is the polynomial ring with infinitely many variables

\[R[y_0, y_1, y_2, \ldots],\]

where the derivation \(\partial\) is extended such that

\[\partial(y_i) = y_{i+1}\quad \forall i \in \mathbb{N}.\]

This method takes as an input a differential polynomial and it computes its derivative. The derivation on \(R\) will be assume with the method derivative. However, the user is allowed to force a particular derivation with the input derivation.

The output of this method is always a differential polynomial. If the user wants the output to be considered as a multivariate polynomial (fixing the amount of variables), the method polynomial from the output do that job.

INPUT:

  • element: the differential polynomial we want to differentiate. This object has to be a polynomial with infinitely many variables.

  • times: a positive integer indicating the amount of derivations to compute.

  • derivation: derivation of the coefficients. If set to None, we use the derivation on the coefficients of the polynomial ring.

OUTPUT:

A polynomial with infinitely many variables on the same ring as element representing the derivation of element.

EXAMPLES:

sage: from ajpastor.dd_functions import *
sage: R.<y> = InfinitePolynomialRing(QQ['a']); a = R('a')
sage: infinite_derivative(y[2]^2 - a)
2*y_2*y_3 - 1
sage: infinite_derivative(a*y[1])
a*y_2 + y_1
sage: infinite_derivative(y[2]^2 - a, derivation=lambda p : 0)
2*y_2*y_3
sage: infinite_derivative(a*y[1], derivation=lambda p : 0)
a*y_2
sage: infinite_derivative(a*y[1], times = 2)
a*y_3 + 2*y_2
sage: infinite_derivative(y[5]^7, times = 0)
y_5^7
sage: infinite_derivative(R.base()(a))
1
sage: infinite_derivative(y[2]^2, times = -1)

ajpastor.dd_functions.toDiffAlgebraic.diffalg_reduction(poly, infinite=False, debug=False)¶

Method that receives a polynomial ‘poly’ with variables y_0,…,y_m with coefficients in some ring DD(R) and reduce it to a polynomial ‘result’ with coefficients in R where, for any function f(x) such that poly(f) == 0, result(f) == 0.

The algorithm works as follows:

1 - Collect all the coefficients and their derivatives 2 - Perform the simplification to each pair of coefficients 3 - Perform the simplification for the last derivatives of the remaining coefficients –> gens 4 - Build the final derivation matrix for ‘gens’ 5 - Create a vector v for represent ‘poly’ w.r.t. ‘gens’ 6 - Build a square matrix –> M = (v | v’ | … | v^(n)) 7 - Return det(M)

INPUT:

  • poly: a polynomial with variables y_0,…,y_m. Also an infinite polynomial with variable y_* is valid.

  • infinite: if True the result polynomial will be an infinite polynomial. Otherwise, a finite variable polynomial will be returned.

  • debug: if True, the steps of the algorithm will be printed.

Method that compute an differentially algebraic equation for the object func.

All differentially definable functions are also differentially algebraic. This method do the transformation from a linear differential equation with difficult coefficients into a non-linear equation with simple coefficients.

ajpastor.dd_functions.toDiffAlgebraic.inverse_DA(poly, vars=None, infinite=False)¶

Method that computes the DA equation for the multiplicative inverse of the solutions of a DA equation.

We assume that the variables given by vars are the variables representing the solution, namely vars[i+1] = vars[i].derivative(). If vars is not provided, we take all the variables from the polynomial that is given.

ajpastor.dd_functions.toDiffAlgebraic.guess_DA_DDfinite(poly, init=[], bS=None, bd=None, all=True)¶

Method that tries to compute a DD-finite differential equation for a DA-function with constant coefficients.

It just tries all the possibilities. It uses the Composition class to get the possibilities of the orders for the coefficients.

INPUT:

  • poly: polynomial with the differential equation we want to mimic with DD-finite elements

  • init: initial values of the solution of poly

  • bS: bound to the sum of orders in the DD-finite equation

  • db: bound to the order of the DD-finite equation

  • all: compute all the possibles equations

ajpastor.dd_functions.toDiffAlgebraic.guess_homogeneous_DNfinite(poly, init)¶

Method that tries to compute a Dn-finite differential equation for a DA-function defined from an homogeneous equation.

This method simplifies the equation supposing that the solution is of the form y(x) = exp(int(u(x))), getting a differential equation for u(x) of less order.

If we end up with a linear equation, then we can return a Dn-finite function where n depends on how many iterations we needed.

If the final algebraic equation is not linear, then we return this last equation together with the number of iterations done.

INPUT:

  • poly: polynomial with the differential equation we want to mimic with DD-finite elements

  • init: initial values of the solution of poly (must be enough).

ajpastor.dd_functions.toDiffAlgebraic.Exponential_polynomials(n, parent)¶

Method to compute the \(n\)-th exponential polynomial.

Assume that \(f(x)\) is a function that can be written in terms of the exponential function in the following way:

\[f(x) = e^{\int_x u(t)dt}.\]

Then we have that for any \(n \in \mathbb{N}\), the \(n\)-th derivative of \(f(x)\) can be written as

\[f^{(n)}(x) = f(x) p_n(u),\]

where \(p_n \in \mathbb{Q}\{u\}\), i.e., \(p_n\) is a differential polynomial. This \(p_n\) is called the \(n\)-th exponential polynomial. This can be computed with a recursive formula:

\[f^{(n)}(x) = (f^{(n-1)}(x))' = f(x)(p_1(u)p_{n-1}(u) + (p_n(u))')\]
INPUT:

  • n: positive integer

  • parent: the resulting polynomial will be included into this parent structure.

OUTPUT:

The \(n\)-th exponential polynomial in the corresponding parent structure.

EXAMPLES:

sage: from ajpastor.dd_functions include *
sage: Exponential_polynomials(1,QQ)
y_0
sage: Exponential_polynomial(2, QQ)
y_1 + y_0^2
sage: parent(Exponential_polynomial(3, QQ))
Infinite polynomial ring in y over Rational Field
sage: parent(Exponential_polynomial(2, QQ[x]))
Infinite polynomial ring in y over Univariate Polynomial Ring in x over Rational Field
sage: Exponential_polynomial(0, QQ)
1
sage: Exponential_polynomial(-2,QQ)
